Assignment Description

 

From chapters 1-6, we learned the basics to JavaScript, where we learned to work with different data types, functions, operators and arrays. We were also introduced to building arrays, controlling flow, error handling, and debugging. We are also introduced to working with the Document Object Model or also known as DOM, DHTML, and enhancing and validating forms

 

In chapter 1, we made basic JavaScript programs so we got to see how basic JavaScript looks.


In chapter 2. we learned how to use diferent data types, functions, and operators.

 

In chapter 3, we learned how to make an array and controlling flow.

 

In chapter 4, we learned about debugging tools and eror handling.


In chapter 5, we are introduced to DOM and how to work with it.

 

In chapter 6, we are shown how to enhance and validate forms





From chapters 7 to 12 We were also introduced to Object-Oriented JavaScript and to manipulating data in strings. Also we are introduced to managing State information and security, programming touchscreens and mobile devices. Lastly, we learned about updating web pages with Ajax and what jQuery is.

 

In chapter 7, we learned about Object-Oriented Javascript programming

 

In chapter 8, we learned how to manipulate data in arrays and strings.


In chapter 9, we learned about security and how to work with state information

 

In chapter 10, we learned how to make programs work with touchscreen and mobile devices


In chapter 11, we learned how to update pages with Ajax

 

In chapter 12, we got an introduction to what jQuery is
